The goal of BibleJourney is to provide an accessible Bible Study experience. Current features include:

  • The Old and New Testament available in WEB (World English Bible) Translation.
  • Note taking
  • Bookmarks
  • Memory Verse Flash Cards

BibleJourney is currently maintained by Jansen Yan. If you'd like to contribute or if you see an error on the site, please feel free to let me know, or visit the Github Repo and submit a pull request. It would be greatly appreciated!
